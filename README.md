# Calculator

## Overview

This project is a simple calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an interactive and user-friendly interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Decimal Calculations
- Clear Button
- Delete Button
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

Calculator/
│
├── index.html
├── style.css
├── script.js

## How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open index.html using Live Server.
4. Start performing calculations.
